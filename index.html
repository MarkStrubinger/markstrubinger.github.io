Node Weather App

HTML, Tailwind CSS, Javascript, Node.js, Express.js, EJS Templating.

A minimalist design weather app, used with the OpenWeather API. The user inputs a city name, and the app returns a page with the current weather information, as well as date and time. This includes Temperature, Heat Index, a Weather Icon (clouds, sun, rain etc.), and a short description. Depending on the weather and time of day, the background will display a different gradient color. Deployed on Heroku.
